Проект: марктеплейс B2B для продажи красок
Способ реализации: Веб сайт на vue js, tailwindCSS, PrimeVue, Lucide Icons, Axios
Пользователи: 
- Посетитель(не зарегистрированный пользователь). Доступный функционал: Регистрация и вход; Просмотр товаров
- Пользователь. Доступный функционал: Работа со своим профилем; Работа с Избранным; Создание компании
- Менеджер компании Доступный функционал: Составление корзины; Создание или отмена заказа; Отслеживание заказа; Чат с Менеджером системы; Работа с заказами компании
- Админ компании. Доступный функционал: Работа со списком Менеджеров компании. Работа с правами Админа компании. Работа с информацией о компании
- Менеджер системы. Доступный функционал: Просмотр заказов компаний. Подтверждение или отмена заказов. Чат с Менеджером компании. Подтверждение или отмена создания компании
- Админ системы. Доступный функционал: Работа со списком Менеджеров системы. Работа с правами Админа системы

Пояснение ролей: Есть компания производитель красок, где работают менеджеры = менеджер системы и обрабатывают в ручном режиме заявки клиентов. Клиенты = менеджер компании, админ компании. Пользователь наследует права от посетителя. Менеджер компании  наследует права от пользователя. Админ компании наследует права от менеджер компании. Менеджер системы наследует права от пользователя. Админ системы наследует права от Менеджер системы

2) Авторизация

9) Подробная информация о заказе

Меню системы пользователя: 
1) О производителе - перенаправление на другой сайт
2) Выход из аккаунта
3) Товары
4) Избранное
5) Профиль
6) Сообщения

Меню системы пользователя: 
1) О производителе - перенаправление на другой сайт
2) Выход из аккаунта
3) Товары
4) Избранное
5) Профиль
6) Сообщения

Меню системы менеджера компании и админ компании: 
1) Заказы
2) Компания
3) Корзина

Меню системы менеджера системы: 
1) Заказы
2) Список компаний
3) Аналитика

Меню системы админ системы: 
1) Заказы

Функционал страниц:
1) Страница остатков и акций:
    - Блок отображения промоакций(прямоугольник под разные баннеры)
    - Карточки товаров из остатков. Карточка товара - картинка с изображением краски, индикатор цвета RAL, название номеклатурное, количество остатка. По нажатию на карточку переходим на страницу подробной информации о товаре
    - 
2) Авторизация
    - Поле для логина
    - Поле пароль с возможностью скрытия и отображения
    - Кнопка восстановления пароля
    - Кнопка перехода на регистрацию
    - Кнопка войти - после успешной авторизации перенаправляем на главную страницу
3) Регистрация
    - Логин
    - Пароль
    - ФИО
    - Повторение пароля
    - Кнопка переход на Авторизация
    - Галочка на согласие обработки данных
    -    
4) Товары
    - Поиск товаров
    - Фильтрация товаров
    - Сортировка товаров
    - Отметить “Избранное”
    - Наличие товара
    - Краткая информация о товаре
    - Для ролей менеджер компании и админ компании: Добавить в корзину
    -    
5) Избранное
    - Просмотр товаров
    - Удаление из избранного
    -    
6) Профиль
    - Сменить ФИО
    - Сменить пароль
    - Указать/сменить почту
    - Настройка уведомлений
    - Создание компании
    -    
7) Сообщения
    - Для роли пользователь: Чат с поддержкой. Приглашения в компанию
    - Для роли менеджер компании: Чат с менеджерами системы по заказу. Чат с участниками компании   
    - Для роли менеджер системы: Чат с менеджерами компании по заказу
    -
8) Заказы
    - Для роли менеджер компании и админ компании: Просмотр информации о заказах. Отмена заказа. Чат с Менеджером системы по заказу. История заказов. Переход к карточке заказа
    - Для роли менеджер системы: Новые заказы, затем активные и потом неактивные заказы.Открыть чат с менеджером компании по заказу. Фильтровать по ОГРН или названию компании (например, если к клиенту назначен конкретный менеджер)
    -
9) Компания
    - Для роли менеджер компании и админ компании: Просмотр участников компании. Просмотр информации о компании. Переход к чату с участниками компании
    - Для роли админ компании: Добавление/удаление участников. Выдача прав администратора. Изменение информации о компании
    -
10) Корзина
    - Просмотр товаров
    - Изменить кол-во товара
    - Удаление товара
    - Добавить в Избранное
    - Оформить заказ
    - Индикатор, будет ли заказан товар из остатка или из новой партии
    -
11) Список компаний
    - Поиск компании по названию или ОГРН
    - Просмотр контактных данных компании
    - Просмотр списка заказов компании
    - Просмотр списка участников компании
    -
12) Пользователи
    - Просмотр списка пользователей: их ролей, принадлежности к компании
    - Удалять пользователей
    -  
13) Подробная информация о товаре
    - картинка с изображением краски
    - индикатор цвета RAL
    - название номеклатурное
    - количество остатка (если переходим с страницы остатков)
    - информация о партии краски (если переходим с страницы остатков)
    - информация о результате анализов краски (если переходим с страницы остатков)
    - ориентировочный срок поставки (если переходим с страницы остатков)
    - кнопка добавить в избранное
    - кнопка добавить в корзину с указанием объема
14) Подробная информация о заказе
    - Статус заказа
    - Список товаров из заказа
    - Менеджер сформировавший заявку
    - Менеджер обрабатывающий заказ со стороны производителя краски
    -
